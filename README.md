# BellasUp

Sistema inteligente de agendamentos para profissionais de estética e beleza.

## Tecnologias Usadas

- React + Vite
- TypeScript
- Tailwind CSS
- Supabase
- Shadcn UI

## Como Iniciar

1. Clone o repositório
2. Instale as dependências com `npm install`
3. Inicie o servidor local com `npm run dev`

Sinta-se à vontade para explorar e modificar o projeto com as suas necessidades da marca BellasUp.
